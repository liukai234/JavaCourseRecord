### ⚡ 命名惯例、wuhu~ 🌈

1. 项目名全部小写
2. 包名全部小写
3. 类名首字母大写,其余组成词首字母大写
4. 变量名、方法名首字母小写，其余每个单词的首字母大写
5. 常量名全部大写
6. 名称不能使用Java中的关键字
7. 名称只能由字母、数字、_下划线、$符号组成
